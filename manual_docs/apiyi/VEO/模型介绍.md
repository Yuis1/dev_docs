# 支持的模型

VEO 3.1 提供 8 种模型变体，满足不同需求：

|            模型名            |      描述      |  价格   |
|---------------------------|--------------|-------|
|          `veo-3.1`          |    默认竖屏视频    | $0.25 |
|        `veo-3.1-fl`         |  竖屏+首尾帧转视频   | $0.25 |
|       `veo-3.1-fast`        |    竖屏+快速     | $0.15 |
|      `veo-3.1-fast-fl`      | 竖屏+快速+首尾帧转视频 | $0.15 |
|     `veo-3.1-landscape`     |     横屏视频     | $0.25 |
|   `veo-3.1-landscape-fl`    |  横屏+首尾帧转视频   | $0.25 |
|  `veo-3.1-landscape-fast`   |    横屏+快速     | $0.15 |
| `veo-3.1-landscape-fast-fl` | 横屏+快速+首尾帧转视频 | $0.15 |

-   `-fl` 代表帧转视频模式（Frame-to-Video）
-   `-fast` 代表快速生成，用时短、价格更便宜
-   默认为竖屏，带 `landscape` 的为横屏


按成功计费： 仅对成功生成的视频收费，生成失败不产生费用

## 生成速度
VEO 3.1 生成速度： 比 Sora 2 快 50%！
快速模型 (-fast)：约 30-60 秒
标准模型：约 1-2 分钟
